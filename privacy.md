# Privacy Policy
## 1. General Information
This app does not collect or process any personal data.

The app can be used without providing any personal information.
 
## 2. Data Processing within the App
The app operates entirely locally on the user's device.

- No data is transmitted to external servers

- No user accounts are created

- No tracking or analytics are used

- No personal data is collected, stored, or processed
 
## 3. Location Selection (Map Functionality)
The app uses Apple’s MapKit to allow users to select a location on a map.

- The selected location is used solely for calculations within the app

- The selected data is not transmitted to the developer

- All processing occurs locally on the user’s device

Apple services may be involved in providing map data. For more information, please refer to Apple’s Privacy Policy:
https://www.apple.com/legal/privacy/
 
## 4. User Data Storage
The app allows users to save and load data in JSON format.

- The storage location is entirely determined by the user (e.g., local storage or iCloud)

- The developer has no access to this data

- Any synchronization via iCloud is handled by Apple
 
## 5. Local Storage of Settings
App settings are stored locally on the user’s device.

- This data remains exclusively on the user’s device

- No data is transmitted to external servers
 
## 6. Internet Connection
The app does not require an active internet connection for its core functionality.

An internet connection may only be required if:

- map data is loaded (Apple MapKit)

- the user utilizes iCloud or similar services
