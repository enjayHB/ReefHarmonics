# ReefHarmonics
ReefHarmonics is an iOS app to simulate sun, moon and resulting astronomical tides for any location at any time using up to 37 harmonic constituents. 
The astromonical computations are based on the "Astronomical Algorithms" by Jean Meeus, 2nd edition (English version). 
Tidal data is computed using relevant formulars from the "Manual of Harmonic Analysis and Prediction of Tides", by Paul Shureman, Special Publication No 98, US Dept of Commerce Coast and Geodetic Survey, Revised (1940) Edition, reprinted 1971 with corrections.
